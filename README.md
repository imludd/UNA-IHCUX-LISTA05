# UNA-IHCUX-LISTA05
Atividade lista 5
Visibilidade do status de sistema: Manter o usuário informado sobre o status do sistema por meio de feedbacks e avisos em tempo real.
Prevenção de Erros: Crição de travas ou confirmações para evitar que o usuário erre antes mesmo do problema acontecer.
Estética e design minimalista: Manter apenas o que é essencial para a tarefa, sem poluição visual.
